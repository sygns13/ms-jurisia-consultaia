REQUISITOS PARA ADMITIR DEMANDA DE TENENCIA 

REQUISITOS DE PROCEDIBILIDAD
MARCO NORMATIVO	REQUISITOS
Artículo 427° del Código Procesal Civil.	
1.	El demandante carezca de legitimidad para obrar 
2.	El demandante carezca manifiestamente de interés para obrar
3.	Advierta la caducidad del derecho
4.	No exista conexión lógica entre los hechos y el petitorio.
5.	El petitorio fuese jurídica y físicamente imposible.
•	Cuando se identifica que no cumple con los requisitos en mención entonces se declara improcedente.

REQUISITOS DE ADMISIBILIDAD
MARCO NORMATIVO	REQUISITOS
Artículo 424 del Código Procesal Civil.	  
1.	La designación del juez ante quien se interpone la demanda
2.	El nombre completo del o la demandante, datos de identidad, dirección domicilio real, procesal, casilla electrónica de acuerdo a la ley 30229.
3.	El nombre completo del o la demandada, datos de identidad, dirección domicilio real; número de celular de ser posible.
4.	Petitorio pretensión clara y concreta de lo que pide
5.	Indicar los fundamentos de hecho del petitorio 
6.	Indicar la fundamentación jurídica del petitorio.
7.	Indicar el monto del petitorio.
8.	Ofrecimiento de todos los medios probatorios.
9.	Firma del demandante
10.	Firma del abogado
Artículo 130 del Código Procesal Civil.	1.	Si el escrito tiene anexos, estos serán identificados con el número del escrito seguido de una letra.
2.	Si el escrito tiene otrosíes o formulas similares, estos deben contener pedidos independientes del principal.
Artículo 425 del Código Procesal Civil.	1.	Copia legible del DNI
2.	Copias certificadas u originales de los medios probatorios de forma legible

Resolución Administrativa N° 481-2025-CE-PJ	1.	Pago de aranceles judiciales por ofrecimiento de pruebas y derecho de notificación.
Se exceptúa cuando están asesorados por la defensa público o por algún estudio jurídico gratuito.
•	Cuando se identifica que no cumple con los requisitos en mención entonces se declara inadmisible.
